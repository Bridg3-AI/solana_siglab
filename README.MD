# AI 기반 위험도 평가 및 마이크로 보험 DApp

## 프로젝트 개요

스마트폰의 센서 데이터와 AI 분석을 활용하여 **실시간 위험도를 평가하고 맞춤형 마이크로 보험**을 제공하는 DApp입니다. Solana Mobile Wallet Adapter를 통해 모바일에서 편리하게 사용할 수 있습니다.

## 주요 기능

실시간 센서 데이터와 AI 분석을 통해 사용자의 위험도를 측정하고, 자동차 및 생활 위험에 대한 맞춤형 마이크로 보험을 제공합니다.

## 핵심 기능

### 1. 위험도 감지 (Risk Detection)
- **AI 기반 위험도 분석**
  - 모바일 센서 데이터 수집 (가속도계, GPS, 자이로스코프)
  - 운전 패턴 분석 (급가속, 급정거, 급회전 등)
  - 환경 위험 요소 분석 (날씨, 교통상황, 도로 상태)
  - 사용자 행동 패턴 학습 및 예측

### 2. 위험 지수 (Risk Index Calculation)
- **실시간 위험도 점수 계산**
  - 종합적 위험도 점수 (0-100)
  - 다양한 센서 기반 실시간 분석
  - 개인별 위험도 프로필 (운전, 생활, 환경)
  - 예측 모델 기반 위험도 산정

### 3. 맞춤형 마이크로 보험 (Micro Auto-Insurance)
- **자동화된 기반 보험 제공**
  - 위험도에 따른 동적 보험료 산정
  - 실시간 보험가입/해지 시스템 구축
  - 보험금 자동 지급 및 처리
  - 블록체인 기반 투명한 처리

## 사용자 시나리오

### 1. 운전자
```
운전 중 AI 시스템이 운전 패턴을 분석하여 위험도 점수를 실시간으로 계산
```

### 2. 보험 가입자
```
위험도에 따라 AI 알고리즘이 보험료를 산정하고 자동으로 보험 처리
```

### 3. 보험금 청구자
```
위험 상황 발생 시 자동으로 보험금 지급 프로세스 시작
```

### 4. 자동 처리
```
모든 과정이 AI 및 스마트 컨트랙트를 통해 자동으로 처리
```

## 보험 상품 종류

### 1. 자동차 보험
- 운전 패턴 및 위험도 분석
- 실시간 보험료 산정 시스템
- 사고 발생 시 자동 처리

### 2. 생활 상해 보험
- 일상 활동 위험도 분석
- 개인별 위험도 프로필
- 생활 패턴 기반 보험

### 3. 여행자 보험
- 여행지 위험도 분석
- 실시간/임시 데이터 수집
- 위험도 변화에 따른 보험료 조정

### 4. 건강 관리 보험
- 건강 상태 모니터링
- 관리 프로그램 연동
- 예방 중심 보험 상품

## 📱 모바일 앱 구조

### 탭 구조 (Bottom Navigation)
1. **홈 (Home)** - 실시간 대시보드 및 위험도 모니터링
2. **보험 (Insurance)** - 보험 상품 브라우징 및 관리
3. **분석 (Analytics)** - AI 위험도 분석 및 리포트
4. **지갑 (Wallet)** - Solana 블록체인 지갑 연동
5. **설정 (Settings)** - 사용자 프로필 및 앱 설정

### 🎯 탭별 상세 기능

#### 1. 홈 탭 (Home Dashboard)
- **실시간 위험도 점수** (0-100 점수 표시)
- **오늘의 위험도 요약** (운전/생활/환경별)
- **활성 보험 현황** (현재 가입중인 보험 카드)
- **최근 알림** (위험 상황, 보험금 지급 등)
- **빠른 액션** (긴급 보험 가입, 사고 신고)
- **AI 추천** (개인화된 보험 상품 추천)

#### 2. 보험 탭 (Insurance)
**상단 카테고리:**
- 자동차 보험 | 생활 상해 보험 | 여행자 보험 | 건강 관리 보험

**주요 기능:**
- **보험 상품 브라우징** (카테고리별 필터링)
- **즉시 가입** (실시간 위험도 기반 보험료 계산)
- **내 보험 관리** (가입중인 보험 목록 및 상태)
- **보험금 청구** (사진 업로드, 자동 처리 상태)
- **보험료 히스토리** (지불 내역 및 예상 보험료)
- **임시 보험** (단기/여행용 보험 빠른 가입)

#### 3. 분석 탭 (Analytics)
- **위험도 트렌드** (일/주/월별 위험도 변화)
- **운전 패턴 분석** (급가속, 급정거, 경로 분석)
- **센서 데이터 시각화** (가속도, GPS 경로, 속도)
- **AI 인사이트** (개인화된 위험 요소 분석)
- **비교 분석** (평균 대비 내 위험도)
- **개선 제안** (AI 기반 위험도 감소 방법)
- **리포트 생성** (PDF 내보내기, 공유)

#### 4. 지갑 탭 (Wallet)
- **Solana 지갑 연동** (Phantom, Solflare 등)
- **잔액 확인** (SOL, SPL 토큰)
- **거래 내역** (보험료 지불, 보험금 수령)
- **토큰 관리** (보험 관련 NFT, 토큰)
- **스테이킹** (보험 토큰 스테이킹 보상)
- **DeFi 연동** (보험 자금의 유동성 공급)
- **QR 코드** (빠른 송금/수금)

#### 5. 설정 탭 (Settings)
**프로필 관리:**
- 개인정보 (이름, 연락처, 운전경력)
- 차량 정보 (차종, 연식, 주행거리)
- 위험도 프로필 설정

**앱 설정:**
- 알림 설정 (위험도, 보험금, 마케팅)
- 데이터 수집 권한 (센서, 위치, 카메라)
- 언어 및 지역 설정
- 보안 설정 (생체인증, PIN)

**데이터 관리:**
- 데이터 백업/복원
- 개인정보 다운로드
- 계정 삭제
